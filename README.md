# User-Behavior-Drop-off-Analysis-E-commerce-Data-
This project analyses user interaction data from an e-commerce platform to understand customer behavior, identify drop-off points in the purchase funnel, and generate actionable business insights to improve conversion rates.


Objectives

Analyze the user journey from product view to purchase

Identify key drop-off stages in the conversion funnel

Understand how factors like product category, price, and time influence user decisions

Provide data-driven recommendations to improve user retention and sales


Dataset:


Source: Kaggle – Ecommerce Behavior Data from Multi Category Store

Contains user-level interaction data including:

User ID

Event type (view, cart, purchase)

Product details (category, price, brand)

Timestamp of interaction



Tools & Technologies:

Python (Pandas, NumPy, Matplotlib, Seaborn)

SQL (data querying and aggregation)

Power BI / Tableau (interactive dashboard)
